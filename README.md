# code-quiz
---
## Acceptance Criteria
```
GIVEN I am taking a code quiz
WHEN I click the start button
THEN a timer starts and I am presented with a question
WHEN I answer a question
THEN I am presented with another question
WHEN I answer a question incorrectly
THEN time is subtracted from the clock
WHEN all questions are answered or the timer reaches 0
THEN the game is over
WHEN the game is over
THEN I can save my initials and score
```


# Purpose

A JavaScript Quiz!! This a multiple choice quiz to test your knowledge in this field.
So go ahead and give it your best shot and see how much JavaScript you really know.
Here's the scoring system
* You have 60 seconds total to finish the quiz
* If answered correctly you get 20 points.
* If answered incorrectly you get no points and a 10 second reduction from the available time.
* When finished with the quiz the time remaning gets added to your final score.
